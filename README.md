# 🍕 AR Pizza Menu Experience

### 🔗 [Live Demo](https://shahazadi-shaguftha-syed.github.io/PizzaAR/)

An immersive Augmented Reality menu that brings your favorite pizzas to life! This project uses **AR.js** and **A-Frame** to create a stunning, interactive dining experience.

## ✨ Features
- **3D Pizza Visualization**: See a realistic 3D model of the pizza right on your table.
- **Interactive UI**: Integrated buttons to view orders, rate the pizza, and more.
- **Dynamic Feedback**: Visual cues for when the marker is detected or lost.
- **Premium Aesthetics**: Clean, modern interface designed for a high-end experience.

## 🛠️ Technology Stack
- **AR.js**: For high-performance marker-based augmented reality.
- **A-Frame**: To build the 3D scene and handle VR/AR interactions.
- **JavaScript**: For the core logic and marker handling.
- **HTML5/CSS3**: For the responsive overlay UI.

## 🚀 Getting Started
1. **Open the App**: Launch `index.html` in a modern web browser (requires camera access).
2. **Scan the Marker**: Point your camera at the pizza marker (found in `/assets/dish-markers/`).
3. **Interact**: Once the pizza appears, use the buttons on the screen to explore the menu.

## 📖 How to Use
- **Marker Detection**: Point your webcam at the **Pizza Marker**. Once detected, a 3D pizza model will appear.
- **Order System**: Use the UI buttons to simulate an ordering process.
- **Rating System**: You can view the order summary and interact with the rating features.
- **Experience**: Move your device around to see the pizza model from different perspectives!

## 📁 Project Structure
- `/assets`: Contains 3D models and marker patterns.
- `/js`: Logic for marker handling, buttons, and data management.
- `index.html`: The main entry point for the AR experience.
- `style.css`: Custom styling for the AR overlay.
